使用npm init -y 初始化项目
安装babel npm install -g babel-cli
跟目录创建.babelrc 文件 根据项目做好配置
使用babel src/index.js -o dist/index.js 命令转换es6 为 es5
在package.json文件中的scripts-->build 替换上面的转换后直接可用npm run build 完成转换
